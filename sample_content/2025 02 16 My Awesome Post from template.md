## Go example:
```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, DSBG!")
}
```

## Rust example:
```rust
fn main() {
    println!("Hello, DSBG!");
}
```

## Python example:
```python
def main():
    print("Hello, DSBG!")

if __name__ == "__main__":
    main()
```

## JavaScript example:
```javascript
function main() {
  console.log("Hello, DSBG!");
}

main();
```
